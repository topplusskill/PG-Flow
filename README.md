**PGFlow** is a full-stack application that demonstrates a **basic social networking system** with JWT authentication, CRUD for posts, comments, and an interaction system.

## üéØ What is it?

An application that demonstrates:

- JWT authentication with route protection.

- Complete CRUD for posts and comments.

- Post and comment like system (with audio feedback).

- Avatar upload via Cloudinary.

- Responsive frontend with Bootstrap 5.

---

## üîê Main Features

### Authentication System
- Registration and login with JWT
- Passwords encrypted with bcrypt
- Protection of private routes
- Tokens with automatic expiration

### Post CRUD
- Create, read, edit, and delete posts
- Personal dashboard
- Public community feed
- Authorization validations

### Interactions
- Likes on posts and comments
- Sound feedback when liking
- Simple comments
- Real-time counters (frontend)

### Extras
- Avatar upload with Cloudinary
- Responsive design with Bootstrap 5
- Organized RESTful API
- Production deployment (e.g., Render)
